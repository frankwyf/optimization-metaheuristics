# リリースノート（日本語）

## リリース範囲

このリリースは、公開用のオープンソース版としてリポジトリを整理するためのものです。

## 変更点

- ルート構成を整理し、試験的なスクリプトを `legacy/` に移動しました。
- `final/` を現行の公開ベースラインとして維持しました。
- 統一 CLI エントリーポイント `scripts/run.py` を追加しました。
- 乱数シード、繰り返し回数、非表示プロットの軽量パラメータを追加しました。
- 中国語・日本語・英語の補助ドキュメントを追加しました。
- 依存関係の導入と構文検証を行う最小 GitHub Actions を追加しました。
- ルートにあった図を `docs/assets/` へ移動し、文書用アセットとして整理しました。
- 公開前に一般的な機密情報パターンを再確認しました。
